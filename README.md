# pickly_privacy
选这个隐私政策

隐私政策
生效日期：2026年6月23日

"选这个"（pickly）重视您的隐私。本隐私政策说明了我们如何收集、使用和保护您的信息。

一、我们收集的信息
1.1 您主动提供的信息
账户信息：当您使用"通过 Apple 登录"时，我们会接收 Apple 提供的您的姓名和电子邮件地址（或隐私转发邮箱）。
选择数据：您在使用 App 进行选择决策时产生的选项内容、选择结果、场景分类等数据。
快捷模板：您保存的常用选择模板。
1.2 自动收集的信息
我们不使用任何第三方分析工具或广告 SDK，不会自动收集您的设备信息、位置信息或使用行为数据。
二、信息的使用方式
我们收集的信息仅用于以下目的：

提供核心服务：帮您进行随机选择决策，展示历史记录和用户画像分析。
多设备同步：在您登录后，将您的选择历史和快捷模板同步到云端，方便您在多个设备间保持一致。
账户管理：识别您的身份，维护您的登录状态。
提供 VIP 服务：记录您的购买状态，为您提供无限选项、完整历史等高级功能。
三、信息的存储
本地存储：未登录时，所有数据仅保存在您的设备本地（UserDefaults）。
云端存储：登录后，您的选择历史、快捷模板和 VIP 购买记录会加密传输并存储在我们的服务器上。
存储位置：服务器位于中国大陆。
数据保留：您的数据在您使用服务期间保留。您可以通过退出登录来停止数据同步。
四、信息的共享与披露
我们不会出售、交易或转让您的个人信息给第三方，除非：

获得您的明确同意。
法律法规要求或政府机关依法要求。
五、第三方服务
本应用使用以下服务：

Sign in with Apple（Apple 登录）：由 Apple 提供，受 Apple 隐私政策 约束。
StoreKit（App 内购买）：由 Apple 处理支付，我们不收集或存储您的支付信息（如信用卡号）。
除此之外，本应用不集成任何第三方分析 SDK、广告 SDK 或推送服务。

六、您的权利
您对您的数据享有以下权利：

访问：您可以在 App 内查看您的历史记录和模板。
修改：您可以编辑您的用户名和模板内容。
删除：您可以在 App 内删除历史记录和模板。退出登录后，本地数据将被清除。
停止同步：退出登录后，后续数据将仅存储在本地。
七、数据安全
我们采取合理的技术和管理措施保护您的信息安全，包括：

使用 HTTPS 加密传输数据。
使用 JWT Token 进行身份认证。
服务器端数据库访问控制。
八、儿童隐私
本应用不针对 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您是家长并发现您的孩子向我们提供了个人信息，请联系我们。
